# zazu-npms

> Zazu plugin to search for npm packages with [npms.io](https://npms.io/)

## Install

Add `danielbayerlein/zazu-npms` inside of plugins block of your `~/.zazurc.json` file.

```json
{
  "plugins": [
    "danielbayerlein/zazu-npms"
  ]
}
```
